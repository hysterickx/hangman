# Hangman Game

A modern desktop version of the classic Hangman word-guessing game. This project showcases dynamic UI component rendering, state validation, custom wordlist handling, and complex conditional state management in Python.

## 🚀 Key Features

- **Automated Clues:** Automatically reveals the first and last letters of the hidden word at the start of each game.
- **Dynamic Gallow Drawing:** Visually renders parts of the gallows and the stick-man step-by-step upon incorrect guesses.
- **State & Life Counter:** Tracks remaining attempts (up to 8 mistakes allowed) with live screen updates.
- **Input Tracking:** Evaluates character matching while preventing repeated inputs and managing game-over or victory states gracefully.

## 🛠 Tech Stack

- **Language:** Python 3.11+
- **GUI Framework:** CustomTkinter
- **Architecture:** Object-Oriented Programming (OOP) with dynamic data filtering.
